### Purpose

The robot testing utility(RTU) is meant to streamline and expand the checks done post-match in the pit.

## How
1. Connect to the robot via deploy cable or wifi (recommended)
2. If you wish to connect via wifi, select the network named “name”.
3. Open DriverStation and Elastic
4. Enable the robot though DriverStation and set the connection type appropriately.
5. On Elastic, press the shortcut “ctrl+D” and select “Elastic-layout-with-pit” and confirm
6. The RTU displays and interfaces are located on the tab titled “CAN/Power” and “Pit commands”.

##### Power 
The power indicator of a device will turn on if the device has at least a 4-volt supply.
i.e., this does not check for brownouts, only that power is being fed to the device.
