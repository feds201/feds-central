# 📖 Programming Cookbook

Welcome to the FEDS 201 Programming Cookbook — a collection of **practical, copy-paste code recipes** for common tasks you'll encounter when building and maintaining our robot code.

<Callout type="info">
  These recipes are pulled directly from the real FEDS 201 codebase. They show you exactly how we do things, with working code you can adapt for your subsystem or project.
</Callout>

## When to Use This Section

- **You need to do something specific** and want working code, not theory
- **You're building a new subsystem** and need the boilerplate
- **It's a new game year** and you need to set up simulation from scratch
- **You're adding automated tests** and want to know the exact pattern
