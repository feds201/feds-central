# 🗺️ Codebase Architecture

The `feds-central` repository is a **monorepo** — everything FEDS 201 builds lives here. This page is your map. Whether you're a new programmer figuring out where things are, or a returning member wondering "where did they put that thing," start here.

---

## Repository Overview

```
feds-central/
├── robot/                      🤖 All robot code (Java, Gradle)
│   ├── 2026-rebuilt/             Current season robot
│   ├── 2025-reefscape/           Last season (archived)
│   ├── sim-core/                 Reusable physics simulation library
│   ├── robot-testing-utility/    Reusable automated test harness (RTU)
│   └── template/                 Starter template for new robot projects
│
├── scouting/                   📊 Scout-Ops scouting platform
│   ├── scan/                     Mobile app (Flutter/Dart)
│   ├── desktop-client/           Admin dashboard (Electron/React)
│   ├── server/                   Backend API (Python/Flask)
│   ├── data-source/              Data pipeline utilities
│   ├── data-chunk/               Data chunking/processing
│   └── old-yearly/               Past season scouting configs
│
├── g201/                       🌱 G201 Eco Calculator
│   └── calculator/               Web app (HTML/CSS/JS)
│
├── operations-manual/          📖 This documentation site (Next.js/Nextra)
│
├── dev-dashboard/              🖥️ Internal developer tools
│
└── game-manuals/               📕 FRC game manual archives
```

## Robot Projects

All robot code lives under `robot/`. Each season gets its own project.

### Current Season: `robot/2026-rebuilt/`

| Aspect | Detail |
|---|---|
| **Language** | Java 17 |
| **Framework** | WPILib 2026.1.1 |
| **Build System** | Gradle (with WPILib plugin) |
| **Drivetrain** | CTRE Swerve (TunerConstants generated) |
| **Simulation** | Dual-engine: MapleSim (drivetrain) + ODE4J (game pieces) |
| **Testing** | RTU (@RobotAction automated tests) |

**Key directories:**
- `src/main/java/frc/robot/subsystems/` — drivetrain, intake, shooter, feeder, climber, LEDs
- `src/main/java/frc/robot/sim/` — simulation classes (RebuiltSimManager, RebuiltField, ShooterSim)
- `src/main/java/frc/robot/utils/` — utilities (LimelightHelpers, SubsystemStatusManager, DeviceTempReporter)
- `src/main/java/frc/robot/commands/` — command classes
- `vendordeps/` — third-party library JSON files (CTRE, REV, PathPlanner, etc.)

**Build & Run:**
```bash
cd robot/2026-rebuilt
./gradlew build           # compile + tests
./gradlew deploy          # deploy to roboRIO
./gradlew simulateJava    # run simulation
```

### `robot/sim-core/` — Physics Simulation Library

A **reusable** physics engine that is game-year-independent. Built on ODE4J.

| Aspect | Detail |
|---|---|
| **Language** | Java 17 |
| **Physics Engine** | ODE4J 0.5.4 |
| **Purpose** | Rigid body physics for game pieces, field collisions, chassis simulation |

**What it provides:** PhysicsWorld, FieldGeometry, ChassisSimulation, GamePieceManager, IntakeZone, LaunchParameters, ScoringTracker, proximity sleep/wake, adaptive sub-stepping.

**Used by:** 2026-rebuilt, 2025-reefscape

For full API docs, see **Simulation System Design** in the FRC section.

---

### `robot/robot-testing-utility/` — RTU Library

A **reusable** test harness that auto-discovers `@RobotAction` methods.

| Aspect | Detail |
|---|---|
| **Language** | Java 17 |
| **Purpose** | Automated hardware testing in Driver Station Test mode |
| **Dashboard** | Embedded HTTP server on port 5800 with Chart.js charts |

**Used by:** 2026-rebuilt (via Gradle project dependency)

For full docs, see **Robot Testing Utility** in the FRC section.

---

### `robot/template/` — Robot Project Starter

A minimal WPILib robot project with swerve drive already configured. Use this as the starting point for a new season:

```bash
# Copy the template for a new year
cp -r robot/template robot/2027-newgame
```

## How They Connect

```
                    feds-central (monorepo)
                          │
          ┌───────────────┼───────────────┐
          │               │               │
       robot/          scouting/        g201/
          │               │               │
    ┌─────┼─────┐    ┌───┼───┐      calculator/
    │     │     │    │   │   │     (standalone web app)
  2026  sim-  RTU  scan dektop server
 rebuilt core      (Flutter)(Electron)(Flask)
    │     │  │        └───┬───┘
    │     │  │            │
    ├─uses┘  │     Bluetooth PAN sync
    └─uses───┘
```

- **2026-rebuilt** depends on **sim-core** (physics) and **RTU** (testing) via Gradle project references
- **scouting/scan** syncs data to **scouting/server** via HTTP, which feeds **scouting/desktop-client**
- **g201/calculator** is fully standalone — no server required
- **operations-manual** documents everything but has no runtime dependencies on other projects
